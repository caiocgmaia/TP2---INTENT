# Cadastro de Livros

## Descrição
Este é um aplicativo Android simples que permite cadastrar livros.  
O usuário digita o **título** e o **autor** na primeira tela e, ao clicar em cadastrar, é levado para a segunda tela que mostra os detalhes do livro.

## Estrutura do projeto

- **MainActivity.kt** – Primeira tela (cadastro de livros)  
- **DetalhesActivity.kt** – Segunda tela (mostra detalhes)  
- **res/layout/activity_main.xml** – Layout da primeira tela  
- **res/layout/activity_detalhes.xml** – Layout da segunda tela  
- **AndroidManifest.xml** – Declara as activities  

## Como testar
1. Abra o projeto no Android Studio.  
2. Compile e rode em um emulador ou dispositivo Android.  
3. Digite um título e um autor e clique em cadastrar.  
4. Confira os dados na segunda tela.
